Birthday Website Pack
=====================

This zip contains a 10-page birthday website rewritten for a deep friendship / best-friend vibe.

Folder contents
---------------
- birthday1.html  -> Birthday greeting
- birthday2.html  -> How you met
- birthday3.html  -> When they became important
- birthday4.html  -> Favorite memories
- birthday5.html  -> How they changed your life
- birthday6.html  -> 25 reasons they are awesome
- birthday7.html  -> Future memories / experiences
- birthday8.html  -> Letter page
- birthday9.html  -> Final surprise
- birthday10.html -> Grand finale
- style.css       -> Shared responsive styling

Assets to add
-------------
Place these image files in the same folder as the HTML files:
- Image1.jpg
- Image2.jpg
- Image3.jpg
- Image4.jpg
- Image5.jpg
- Image6.jpg
- Image7.jpg
- Image8.jpg
- Image9.jpg
- Image10.jpg

How to use
----------
1. Extract the zip.
2. Put your photos into the same folder with the filenames above.
3. Open birthday1.html in a browser.
4. If you host on GitHub Pages, upload all HTML files, style.css, and images together.

Notes
-----
- The site is mobile-friendly and responsive.
- The image frames use object-fit: contain so the full image is visible.
- If you want the images to fill the frame instead of showing the full picture, change object-fit: contain to object-fit: cover in style.css.
